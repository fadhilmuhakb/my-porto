Inventory Application

This inventory application was created to meet the needs for storing field equipment databases and managing the equipment borrowing process. This application was developed using Laravel technology as a backend framework and jQuery to make it easier to manage interactions on the frontend side. The final result of this application is a loan report and information regarding the number of tools available.
