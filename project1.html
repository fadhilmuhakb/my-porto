Absence Application

This attendance application functions to retrieve data from the attendance tool. The data is then exported in Excel format, imported into the application, and processed to produce the required information. This application was developed using Laravel technology as a backend framework and jQuery to make it easier to manage interactions on the frontend side. The final result of this process is an Excel file containing attendance data that has been processed according to requirements.
