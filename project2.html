Accounting Application

This application is specifically designed for recording and managing accounting data in companies. Built with Laravel technology as a reliable backend framework and Vue.js to create an interactive and responsive frontend interface, this application makes bookkeeping, tracking transactions and preparing financial reports easier. The output from this application is in the form of a ledger and trial balance, which can be exported in Excel format for easy analysis, storage and sharing of data. The resulting Excel format also allows users to make adjustments or further processing according to company needs.
